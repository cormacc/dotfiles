* Description
Personal additions on top of Spacemacs' built-in =+lang/org= layer:
custom org-mode tweaks, archive helper, web tools, and =org-roam=
setup.

** Features
  - Hierarchical archive (=org-archive-subtree-hierarchical=)
  - org-web-tools for web capture
  - org-roam (with custom capture templates)
  - tasks-org helper from =agents-src/emacs/tasks-org= for org-tasks capture, selection, and plan navigation

* Install
Add =org-user= to =dotspacemacs-configuration-layers= in your
=~/.spacemacs=.

* Key bindings

Layer-local org-mode bindings:

| Binding | Command |
|---------+---------|
| =SPC m g a= | =tasks-org-toggle-task-and-plan= |
| =SPC m g A= | =tasks-org-toggle-task-and-plan-other-window= |
| =SPC m SPC= | =tasks-org-toggle-selected= |

All other layer bindings are inherited from Spacemacs' =+lang/org= layer.
