* Global keybindings

These are the default menus from `agents/pi/extensions/keybindings/defaults.json`.

** `,` — pi agent

| Key | Label | Action |
|-----+-------+--------|
| `, m` | Switch model | `passthrough:ctrl+l` |
| `, t` | Cycle thinking | `passthrough:shift+tab` |
| `, o` | Toggle tool output | `passthrough:ctrl+o` |
| `, k` | Toggle thinking | `passthrough:ctrl+t` |
| `, e` | External editor | `passthrough:ctrl+g` |
| `, n` | New session | `command:/new` |
| `, r` | Resume session | `command:/resume` |
| `, b` | Session tree | `command:/tree` |
| `, c` | Compact | `command:/compact` |
| `, y` | Copy last response | `command:/copy` |
| `, s` | Settings | `command:/settings` |
| `, R` | Reload extensions | `command:/reload` |

** `Space` — leader

| Key | Label | Action |
|-----+-------+--------|
| `Space t E e` | Emacs mode | `keybindings:set-mode-emacs` |
| `Space t E v` | Vim mode | `keybindings:set-mode-vim` |

* Extension suggestions

These bindings are registered via `suggestKeybindings()` from extension code.

** Tasks (`agents/pi/extensions/tasks`)

| Key | Label | Action |
|-----+-------+--------|
| `Space t t` | Show tasks | `command:/tasks` |
| `Space t n` | New task | `command:/tasks new` |

** Git diff (`agents/pi/extensions/git-diff`)

| Key | Label | Action |
|-----+-------+--------|
| `Space g d` | Diff toggle | `command:/diff toggle` |
| `Space g f` | Diff focus | `command:/diff focus` |
| `Space g j` | Diff scroll ↓ | `command:/diff scroll-down` |
| `Space g k` | Diff scroll ↑ | `command:/diff scroll-up` |
| `Space g e` | Diff fold | `command:/diff fold` |

** Term (`agents/pi/extensions/term`)

| Key | Label | Action |
|-----+-------+--------|

| `' f` | Focus | `term:focus` |
| `' h` | Prev session | `term:prev` |
| `' l` | Next session | `term:next` |
